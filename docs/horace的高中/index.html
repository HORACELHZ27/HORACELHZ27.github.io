HORACE的高中

这篇文章主要介绍我的初中及高中，希望通过这来纪念我的青春时光和奋斗的历程

January 28, 2023

中学时光：

由于西安市的小升初及中考制度，我在西安市第三中学及其初中部西安尊德中学度过了六年的时光，这是极其宝贵并且塑造品格的六年时光，正如我的校徽所言：尊道贵德

这里可以通过百度词条来了解西安市第三中学西安市第三中学网址.

校训

尊道贵德： 于右任先生亲手所写的校名尊德中学，取意为尊道贵德:

我的收获：

  在这里我度过了从12岁到18岁这一段人生中最美丽的时光，遇到了初中时的班主任：张娟丽老师、语文老师：张敏老师、英语老师：甘露老师、历史老师：武颖华老师等许多位对我帮助十分巨大的老师，还有我人生中第一个喜欢的女孩、四位玩到现在的hxd、和许多小伙伴们。高中时遇到了班主任：王小英、物理老师：符云辉、数学老师：安雅男等许多位对我帮助也十分巨大的老师，也收获了5位志同道合的小伙伴。这六年生活让我的品格得到了塑造：人要老实、善良；要有自己的坚持；有所为有所不为；要不断向前走、向前出发……这是一所有着120余年历史的中学，这里有丰富的历史底蕴和优秀的教师们，更是走出了一位位优秀杰出的校友：樊登、冯仑、李华星、方东等。希望更多学弟学妹在这里找到自己人生的志向、塑造自己的品格。
